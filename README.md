<h1 align="center">Vue & Vite Boilerplate</h1>

## Under the hood

### Core
- Vue 3
- Vue Router 4
- Vuex 4
- Vite 2

### Style
- SCSS
- PostCSS
- Autoprefixer

### Code Style
- Eslint
- Prettier
- Standard

### Commit convention
- Angular's commit convention

## Usage

Install dependencies:

```
yarn
```

Start dev server:

```bash
yarn dev
```

Build for production:

```bash
yarn build
```

Start preview server:

```bash
yarn serve
```
